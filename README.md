# SwipeIQ

Structure initiale professionnelle du projet SwipeIQ.

## Dossiers
- `mobile/`: application Flutter
- `content/`: ressources éditoriales, assets et scripts de contenu
- `tools/`: outils utilitaires pour le projet
- `infra/`: configuration infra/ops
- `docs/`: documentation technique et produit

## Branching
- `main`: stable
- `dev`: intégration
- `feature/*`: nouvelles fonctionnalités
